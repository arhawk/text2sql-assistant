# App Layer

This directory contains the Streamlit UI for the current lightweight Text2SQL prototype.

Run it with:

```bash
source .venv/bin/activate
streamlit run app/streamlit_app.py
```

If `streamlit` is not installed yet, create the local repo venv first:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
```

The original A2/A4 course snapshots live in [`legacy/`](../legacy/README.md).
