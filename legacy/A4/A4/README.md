# Nested A4 Workspace Copy

This directory is the preserved inner `A4/` workspace from the original assignment tree.

Use this copy when you want the exact original path layout from the source workspace.

## What is inside

- `build_dataset.py`: the original ATIS preprocessing script.
- `Classification/`: the original template-classification experiments, including `a4_experiment.py`.
- `Generation/`: the original seq2seq generation experiments.
- `processed_data/`: the precomputed splits and template mapping files.
- `data/`: the raw ATIS source files used by the original pipeline.

The outer [`legacy/A4/README.md`](../README.md) explains how this nested copy relates to the flattened snapshot.
